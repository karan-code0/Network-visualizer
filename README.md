# Network Visualizer

## 🚀 Real-Time Network Traffic Visualizer

### 🔧 Features:
- Live packet sniffing
- Incoming & outgoing traffic tracking
- Protocol distribution (TCP, UDP, ICMP)
- Real-time graphs (Bar, Line, Pie)
- IP geolocation map

### 🛠 Tech Stack:
- Python
- Scapy
- Plotly
- Dash

### ▶ How to Run:
pip install -r requirements.txt
python app.py
